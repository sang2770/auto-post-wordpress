Auto Store Creator

Automatically create WordPress stores from Google Sheets data

Configuration

No API key required! Just publish your Google Sheet to the web.

Store Sheet: Contains store names and coupon data (store name, coupon name, coupon_code, discount_value, etc.)
Store Details Sheet: Contains store information (name, descriptions, about, guide, q_and_a)
Setup: 1. Open each Google Sheet → 2. "Tệp" → "Chia Sẻ" → "Xuất bản lên web" → 3. Choose "Toàn bộ tài liệu" and "Trang web" → 4. Click "Xuất bản" and paste each URL above
